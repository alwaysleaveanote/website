Links and Pictures

Links

To add a hyperlink to something use anchor tag < a href = "URL" > words to be linked < /a >. Pretty simple.
Note that tinyurl.com can make your html a lot prettier.
Here's a funny video!

you can also link to other pages in your websites file directory using the same method < a href = "FILE_NAME.html > words to be linked < /a>
For example, here's a link back to the homepage.

You can not inherantly tab through picture, you have to provide a tabindex attribute to an anchor tag.
The syntax is tabindex = "1/2/3/etc."

You can also add a linktitle attribute to an anchor tag if the text your anchor encloses doesnt sufficientoy
explain where the link will take you

There is also a accesskey attribute, so when the user presses ctrl + accesskey, it will take them to that link

using the < a href = "#content > tag will skip to the < section > of your page with the attribute "content"

Pictures added in a similar way to links. < img src = "URL OR LOCATION_IN_FILE_DIRECTORY" >

Videos

You can imbed videos much the same you you imbed picture, except instead of the < img > tag, you use the < video > tag.
There is also the control attribute (which allows the user to choose whether or not to play the video) and the
autoplay attribute (which plays the video automatically. Because some people are dicks.) videos also have height
and width attributes, which initialize to the original videos height and width.

NOTE: This does not work with youtube videos (like the one below). You have to copy the embed code under share.

Sometimes specific types of videos don't work with specific browsers. In this case, you can create more than one
source for the video which the browser will try. Looks like < video > < source src = "source" > < /video >

Audio

Audio is the same as video except with the < audio > tag.
