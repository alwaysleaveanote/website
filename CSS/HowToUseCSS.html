How to use CSS

CSS attributes are declared the same way HTML attributes are, except they use colons instead of =.

In Line Styles

These are similar to HTML tags, except they have a style attribute. For example:
< p style = "color: red" > text

gives

text

This is an ugly way to do things, though, so it should be avoided wherever possible.

Internal Styles

These are declared in the < head > section of your file and they give every specified element a
specific attribute. For example :
< head >
< style >
p {
color: red;
}
< /style >
< /head >
makes every paragraph red.
a {
color: blue;
}
makes every link blue.
Still, though, this makes your HTML file ugle, so you usually want to separate the CSS file and the HTML file.
This is where External styles come in

External Styles

External styles look the same as internal styles, except they are saved in a separate .css file and only
contain the "element { styleType }" part. If you save this .css file in the same directory as the HTML file
that will use it, you can reference it in the head section with < link rel="stylesheet" href="style.css" >

Specificity

If you have more than one conflicting CSS rules for the same tag, the computer follows the rule that
more specific = greater presedence, otherwise last one takes presedence. ids have a specificity value of 100,
classes has a value of 10, and HTML tags have a value of 1.

At-rules

at rules use the "@" sign to import other CSS files, apply CSS to particulare mediums, or embed new fonts

@import

@import url(URL) allows you to add more than one CSS stylesheet to your page. This is useful for complex pages where more
than one stylesheet is cleaner and easier to read than just one. These must be placed at the top of your HTML file

@media

@media { selector { attributes } } can be used to define attributes for different types of media. This includes
properties screen, print, projection, handheld, and all. This is useful for making a mobile webpage or modifying
your page for printing

@font-face

The font-face at rule can be used to make even non-web-safe fonts useable on your page. It is used like
a selector, except it essentially just creates a useable font variable. For example, @font-face { font-family: name
src: url(URL) } embeds a font in your webpage so you can use it from then on. You can also add other front properties
to your @font-face

Attribute Selectors

With attribute selectors, you can effect any element with a specific attribute using element[attribute] { ... }.
For example, abbr[title] { border-bottom: 1px dotted } makes a dotted line under any abbreviation with a title attribute.
You can also title only elements with attributes of a specific type (useful for input elements) using element[attribute=type]
attribute ^= something effects an element with an attribute beginning with the specified thing, attribute $= effects
an element with an attribute ending with the specified thing, and attribute *= effects an element with an attribute
containing the specified thing.
