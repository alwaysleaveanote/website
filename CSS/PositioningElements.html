Margins and Padding

margin and padding are both properties. margin determines the space outside something whereas padding determines
the space inside something. You can visualize margins and paddings with the box model. You can also declare margin-top, padding-top, etc. This box model can be applied to every element on a page.

There are also shorthands for margins and paddings. { margin: 10px 5px 2px 1px } makes margin-top = 10px
margin-right 5px, margin-bottom 2px and margin-left 1px. { margin: 10px 5xp } makes margin-top and margin-bottom 10px
and margin-left and margin-right 5px.

Display

display is a selector which usually have the values inline, box, or none. Inline makes elements display
with normal line flow. block gives an element its own "block" of space, but it can also be declared inline
for a more dynamic use. none makes the element invisible. display: none gets rid of the element from the flow
entirely, while visibility: hidden simply hides the element, leaving it in the flow.

Positioning

the position property declares where an element is in reference to where it would be in the normal
flow of the page. There are four values the position property can have: static, fixed, relative, and absolute.
This property allows elements to overlap. You can change the overlapping order by using the z-index property.
The element with a lower z-index will be behind the element with a higher z-index. If a z-index is not specified
the element which is created last will be on top.

  • static: positioned according to the normal flow of the page. not affected by top, bottom, left,
    and right properties
  • fixed: the element does not move even when the page is scrolled
  • relative: positions the element relative to where it would be in the normal flow of the page
  • absolute: positioned relative to the first parent element that has a position value other than static.
    the other elements in the page ignore this element. It does not effect the flow of the page.

After you declare a position property, you can declare a top, bottom, left, or right property which can have values
equal to a length or percentage, which determines how far from the top/bottom/left/right of the page the element is.
