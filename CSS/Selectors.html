Selectors

Where HTML has tags, CSS has selectors. These determine the layout of particular sections of your page
You can give multiple elements the same style attribute by saying element1, element2, etc. { }

HTML Selector

HTML Selectors are a subclass of selectors which are simply the names of HTML tags and are used to change
the style of that tag. the inside of a selector is of the form property: value;. For example:
< body >
fontsize: 14px;
color: navy;
...
< /body >
gives the given value to the given property inside the body selector.

CSS Units

There are certain "units" which are used for many of values.

  • px: unit for pixels (size is still changed when zoomed in or out)
  • em: unit for calculated size of a font. e.g 2em means words are twice the current size
  • pt: unit for point. Usually used in printed media.
  • %: percent...
  • Other units include pc (picas) and SI units.
  • 0: You don't need to include a unit on properties with a 0 value.

Class and ID Selectors

You can declare your own selector using class or ID selectors. You declare style for these selectors
the same way you declare it for an HTML selector, except for ID selectors you use # infront of the name and
for class characters you use . infront of the name. You start an ID section by opening a new div tag and giving
it an id attribute equal to the ID selectors name. You declare a class by giving an ordinary HTML tag a class attribute
equal to the name of the class selectors name.
ids can be used to identify one element whereas classes can be used to identify more than one element. e.g you can
only have one section named to a particular id, but you can have multiple elements named to a particular class.
You can also apply a selector to a particular HTML tag by using HTML_TAG.CLASS { }.

You can also nest HTML, class, and id selectors by creating an id selector (the outermost "section" selector)
then creating a "id HTML/class { }" selector. This makes everything with a certain HTML tag or class in the given
id section the style aspects

Pseudo Classes

Pseudo classes are bolted on to selectors to specify the style of something with a specific state .
It takes the form of selector:pseudo_class { property:value; }.
For example, if you want to make unvisited links blue and visited links purple, you would do a:link { color:blue }
and a.visited {color.purple }. Other pseudo classes included active (something that is activated by the user, such
as a link is clicked), hover, and focus (when you are inside a text box for example).
NOTE: focus is not supported in all browsers, so don't use it for anything important.

There is also the first-child pseudo class which effects whatever the first element of a section is

Pseudo Elements

Much like pseudo classes, pseudo elements are to specify the style of an imaginary sub-element of an element.
There are a number of often used pseudo elements

  • first-letter: the first letter of a heading/paragraph/etc.
  • first-line: the first line of a heading/paragraph/etc.
  • before/after: puts something before/after a given element

Positioning

The position property is used to define whether a block is placed absolute, relative, static (normal), and
fixed (stays at that position on the page regardless of scrolling).

Universal, child, and adjacent selectors

Universal

using the * selector, you target everything. This is commonly used to change things like margin.
The universal selector can also be nested.

Child

the > symbol between two selectors means that the second selector gets the defined attributes iff it is
a direct child of the first selector.

Adjacent

the + symbol between two selectors means that the second selector gets the defined attributes iff it directly
follows the first symbol in your HTML file.
