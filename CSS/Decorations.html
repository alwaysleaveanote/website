Colors

Colors can be defined in 5 ways:

  1. rgb(number, number, number) (0 < number < 256)
  2. rgb(%, %, %)
  3. predefined color (e.g red, blue, navy, teal, etc.)
  4. 3 digit hexadecimal form
  5. 6 digit hexadecimal form
NOTE: You can define color and background-color

CSS also allows you to change the alpha transparency and HSL (hue, saturation and lightness) values.
to define the alpha transparency value, instead of doing color: rgb(red, green, blue) you do
color: rgba(red, green, blue, alpha). You can also define colors using color: hsl(hue, saturatio, lightness) or
color: hsla(hue, saturation, lightness, alpha). You don't need to define hsl and rgb.

Borders

To make a border around an element, all you need is a border-style. This can take the values of solid, dotted,
dashed, double, groove, ridge, inset and outset. You can change the color of the border with border-color.
You can change the width of the border with border-width or border-top-width/border-left-width/etc.

You can also combine border-width/color/style into { border: width color style }. This also works for border-top, etc

Backgrounds

  • background-color: The color... of the background...
  • background-image: The location of the image: url(URL)
  • background-repeat: how the image repeats itself across the background
    repeat (tiles image) repeat-x, repeat-y, no-repeat
  • background-position: top, center, left, bottom, right, any combination, a length, or a percentage

NOTE: You're able to define more than one background by putting them in a comma separated list.

Rounded Corners

Circles

Use the border-radius attribute. You can target specific corners using border-top/bottom-left/right-radius
of border-radius: 4 values (going clockwise from top left) OR 3 values (top left, everything else), OR 2 values
(top left, bottom right, everything else).

Ellipse

Instead of defining circles, you can also define elipses using border-radius: horiz radius/ver radius

Shadows

To define a shoadow for a box you use box-shadow: horizontal-offset vertical-offset blur-radius spread-distance color
horizontal and vertical offset are how far the shadow is nudged right/up. blur radius defines how sharp the shadow is
lower = sharper. spread distance is how large the shadow is. You can also make shadows inside the box using the inset property. There is also a text-shadow attribute, which is the same as box-shadow except it effects text.

Basic Animations

CSS allows you to add basic animations to specific transitions. You can use transition-property (which
properties will be transitioned), transition-duration, transition-timing-function (acceleration function),
and transition-delay. These can all be put into the shorthand transition: . This has to be combined
with different states of the element. For example, for a link to transition, you have to have the base state
(unclicked) which contains the transition attribute, and a different state (hover, etc) with different attributes.
You can also make different rules for different changing attributes. For example, you can make one transition attribute
for color and one for text size and background.

NOTE: Transitions don't work in all browsers. To add a few more browsers, redeclare your transition attribute with
-webkit-transition name.

Transform

The transform attribute allows you to rotate, skew, scale, and translate a box.

Rotating

transform: rotate(xdeg) will rotate everything in your box x degrees.

Skewing

transform: skew(xdeg, ydeg) will tilt everything x degrees horizontally and y degrees vertically

Scaling

transform: scale(number) will multiply the size of the box and everything in it by the number. You can also
define horizontal and vertical scaling with scale(number, number).

Translating

transform: translate(number1, number1) will translate everything number1 right and number2 down. Similar to
position: relative; left: 100px; top: 200px.

Matrices

You can also define a matrix function which rotates, skews, scales, and translated. It takes in 6 numbers

Origin

You can also define where the origin for your transformation is in x and y coords (or use top, bottom, left, right).

Gradients

Linear Gradient

Use the background or background-image property and define a linear-gradient() value. This can contain 2 colors
for an even fade from one to the next from top to bottom. If you want your colors to start and end in different places,
you can add a to direction (or more than one direction) where the second color will end. Instead of to direction
you can also use xdeg to make it fade at any angle. You can also have more than one color.

NOTE: this doesn't work in all browsers, so you will also want to add a background: -webkit-linear-gradient() with the
same information as background: linear-gradient(). webkit doesn't use the to keyword, just the direction.

Radial Gradient

Radial gradients are defined in much the same way as linear gradients, except they start in the center of the box
and spread elliptically to the edges. You can change the way the spread looks in the same way you could change
the way the spread of a linear gradient looked, except you use a shape keyword (usually circle) or where the first color
starts ( at direcction ).

You can also define where each color starts by adding a color stop - a distance or percentage directly after
the color declaration (no comma).

Repeating Linear/Radial Gradient

A repeating linear gradient must be used in conjunction with a color stop and starts over again after the
the last color stop
