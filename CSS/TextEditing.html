Fonts

Fonts are determined with a number of properties:

  • Fonts are defined with the font-family property and have values equal to fonts you usually see on
    your computer (e.g ariel, helvetica, serif). Note: fonts with more than one word in the title are surrounded
    by quotation marks, have spaces, and are capitalized.
  • The font-size property changes the font size, but should not be used in place of header tags.
  • font-weight determines how bold a word is. It's values include lighter, normal, bolder, bold 100, 200, ..., 900 (400 = normal, 700 = bold).
  • font-style states whether the words are italic or not (italic or normal values).
  • text-decoration states whether the words have a line running over, under, or through them
    (underline, overline, line-through values).
  • text-transform determines capitalization (capitalize (first letter of every word capitalized), uppercase,
    lowercase, none

You can also combine font-style/weight/size/height/family into one long font property. e.g { font: style weight
size height family; }. You can do this in any order.

Text Spacing

  • letter-spacing and word-spacing determine the spacing between letters and words and can have
    values equal to a length or normal
  • line-height determines the height of a line without changing the font or text size. Can have values
    equal to a number (specifying a multiple of the font size), a length, a percentage, or normal.
  • text-align determinees where the text is on the page. Can have values of left, right, center, or justify
